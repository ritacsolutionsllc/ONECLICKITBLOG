# ONECLICKITBLOG

Tech blog built with Next.js 14 + Sanity v3. See [CLAUDE.md](CLAUDE.md) for architecture details.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Sanity Studio is at [http://localhost:3000/admin](http://localhost:3000/admin).

## Environment Variables

Copy `.env.example` to `.env.local` and fill in your Sanity project credentials.
